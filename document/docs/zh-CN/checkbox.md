
### Attributes
| 参数        | 说明              | 类型    | 可选值 | 默认值 |
| ----------- | ----------------- | ------- | ------ | ------ |
| clearable   | 是否可清空        | boolean | —      | true   |
| placeholder | 占位文本          | string  | -      | -      |
| label       | 标签文本          | string  | -      | —      |
| name        | 表单对象绑定的key | string  | -      | -      |
| disabled    | 是否禁用          | boolean | -      | false  |
| options     | 多选框内容的数组  | array   | -      | -      |
| className   | 多选框类名        | string  | -      | -      |
| border   | 是否显示边框        | boolean  | -      | false     |
